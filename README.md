# Игра «5 БУКВ» (Wordle на русском)

Это десктопная реализация игры по типу Wordle с поддержкой русского языка. Пользователь пытается угадать загаданное 5-буквенное слово за 6 попыток. Интерфейс реализован с помощью библиотеки `tkinter`, а словарь загружается из файла `слова.txt`.

Весь проект упакован в Docker-образ с GUI-поддержкой, что позволяет запускать игру **на любой системе с Docker без установки Python или зависимостей вручную**.

---

## Структура репозитория
5-bukv/
   Dockerfile # Описание образа Docker
   README.md # Этот файл
   main.py # Основной скрипт игры (ранее wordle.txt)
   слова.txt # Словарь 5-буквенных русских слов
   .env # (опционально) файл переменных окружения

## Требования

- Установленный Docker
- Возможность запуска GUI-приложений из контейнера:
  - **Windows/macOS**: достаточно Docker Desktop с включённым GUI-режимом.
  - **Linux**: требуется доступ к X11 (см. инструкции ниже).

## Cборка и запуск

### 1. Сборка образа

docker build -t 5bukv .

### 2. Запуск приложения 
 На Windows или macOS:
 
docker run --rm -it \
  --name 5bukv-app \
  -e DISPLAY=host.docker.internal:0 \
  5bukv
 На Linux:
Разрешите доступ к X11-серверу и запустите контейнер:
xhost +local:root

docker run --rm -it \
  --name 5bukv-app \
  -e DISPLAY=$DISPLAY \
  -v /tmp/.X11-unix:/tmp/.X11-unix \
  5bukv
# Использование .env (опционально)
На данный момент переменные окружения не используются, но структура поддерживает их в будущем. Если понадобится:

docker run --rm -it --env-file .env 5bukv
